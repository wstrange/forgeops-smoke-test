ForgeOps Smoke Test Service

Run a series of tests against a target deployment of ForgeOps.

You can use curl or REST to run the smoke test. For example:

    curl 'https://smoketest.forgeops.com/test' \
      --data 'fqdn=nightly.iam.forgeops.com&amadminPassword=p3tqhd07v4u6aekvhivj952ffionwwf1'

Or use this form for manual testing (this just does a POST to the /test endpoint):





Note: This is provided to test development or QA instances. Do not post production credentials to this service.
